Project Page

The Project Detail Page is where all details about a project can be viewed. From schedule dates to calculated metrics, everything you need to know about your project is here.

The navbar has a segmented tab selector. By default, when you open a Project Details page, you will be on the Project tab. Each tab contains different information about your project, so make sure you check each of them out.

  • Project Tab
  • Quality Tab
  • Schedules Tab
  • Risk Tab

Top Row​

Regardless of which tab you are on, there is a top row of components. This top row remains constant, except when the project you are on has risk metrics enabled. In this case, the top row will change to the Risk Enabled Top Row only on the risk tab.

Project Picture​

To add some style, be sure to add an image to your project by clicking the image card in the top row. By default, the image is a 'Rhino at Work' sign. Once you click anywhere in this card, a window will popup asking you to upload an image. Once you select an image file, a preview will pop up to the right that will allow you to crop the image to your desired size. The image can only be cropped into a square because the display in the top row card is a 200x200 pixel square. Once you are satisfied, just press save and viola! Your new project image now shows up in the top row. Project images can be edited anytime using the same steps.

tip

Adding a picture of the site (or planned finish look) to a project is a great way to show your clients you are invested in their project when utilizing the Analytics Dashboard in client meetings.

Important Dates​

Just to the right of the project image, you will find a four-square outlining some important dates for this project, all based on the latest update uploaded to this project. The first square lets you know which update this project is on as well as the data date of that update. The second square shows the contract completion date along with the variance it has to the baseline's completion date. The third square displays the current completion date as well as how many days are left until that date. The final square shows the previous update's current completion date, as well as the variance between that one and the current update's completion date.

Scores and Trend Chart​

The remainder of the top row is all telling you the same information that the Portfolio Page's Portfolio Summary tells you, except this component is all about this specific project. When you navigate through different tabs, the most relevant score to that tab will by default be selected.

Risk Enabled Top Row​

For risk enabled projects on the risk tab, this special risk top row will replace the default one. The project picture is still leading the row, but the rest is different from the default top row. Each of these components is similar to one found in the Portfolio Page Risk Analysis Banner, but with the scope of these components being only for this project.

  • Predictability Trending
    • This shows the Predictability Score trending throughout each update of the project instead of how the Portfolio Page does the last 6 months. Hover over an update to see an exact score value.
  • Categories of Risk
  • Heat Map

The greatest strength of these widgets in the risk enabled top row is that you can see the Risk Register just below them so that you can figure out which risks contribute to which part of each component.
