Reports

Gantt Overview​

The Reports Page features Gantt data for all activities from the most recent update. On the table to the left there are a variety of important columns that can be configured in the Column Selector. To the right there are gantt bars that greatly helps visualize the timeline of your project's activities. The red line going through the gantt timeline shows the Data Date of the most recent update.

Filter Menu​

This button opens a window that allows you to apply multiple layered filters to your project. Add multiple filters within filter groups to really nail down the specific activities you are looking for. These filter options are also included as part of the Report Form, where we go over an example of the filters in action.

Group By​

An individual window that can be used to apply groupings to your project's activities by either activity codes or wbs. Further description and images are in the Report Form

Choose Columns​

Column selector for fields visible in the gantt chart's left pane. It is also available as an option in the Report Form.

Report Center​

The Report Center is the hub to manage your report templates and batch-export to PDF multiple reports or even entire folders. The buttons at the bottom of the window allow you to:

  • create new folders (personalized collection of reports)
  • create new reports (templates used to display different activities/relationships in the gantt chart)
  • quick save (save currently displayed gantt layout as a new report template)
  • Add multiple reports to a folder (Requires report checkbox selection)
  • export to pdf (batch export all selected reports to your email after a few minutes. Requires report checkbox selection)

Report Window​

The Reports window combines all transformation options for the gantt chart into one window to be used as a form. You can press the apply button to see your changes first before saving your report or save the report to add it to this project's Report Center. If you got to this window through an edit icon click, clicking save will update that report's values instead.

The general options require a name for the report for your own identifying purposes (reports are tied to projects, so everyone with access to the project can see the report name). Additionally, there are options to base this report on previously created reports (functions like the duplicate button) and 2 management options: Add to Favorites and Add to Folder. These two are more for organizational purposes to either make this report show up in the Favorites Menu or inside a specific folder in the Report Center respectively.

Relationships Toggle​

Simple toggle to show/hide the relationships arrows on the gantt bars with the default being hide.

Criticality Toggle​

Simple criticality toggle with options to display all activities or only critical activities with critical being the default.

Visible Columns​

The same column selector as seen here. By default, the first 7 columns will be checked and visible. All columns, except for the activity ID, can be unchecked to hide or checked to display.

Filters​

A 'Gantt Filters' window will appear when this button is clicked that can help narrow down your chart to see the activities you desire. You can add as many groups which consist of as many filters as needed. The buttons 'And' or 'Or' can be selected for each group which will allow you to create powerful and specific filters to match your ambitions.

info

For this example from the above screenshot, the filtered activities that would show up after hitting apply will be activities that either have an ID equal to 'SUB-1170' OR have a status equal to 'Complete' AND an activity name equal to 'Project Complete'. As you can see, utilizing groups and changing the and/or logic lets you create filters that can showcase complex subsets of activities, allowing you to quickly make visual aids for meetings and presentations.

Grouping​

You can group activities by WBS or by Activity Codes, so you can easily view all your tasks in the format you prefer. Simply select either 'WBS' or 'Activity Codes' to enable your desired grouping method.

You may collapse or expand any of these sections to help visualize how each group is performing. The grouping row is distinctive by its dropdown icon and highlighted color. You can view aggregate values for all activities within the group such as overall start and finish dates for the entire group, or the total remaining duration. We provide a default color scheme for easier visibility to differentiate grouping levels but you can easily customize these colors for you project by clicking on the color and selecting a new one in the grouping section.

Grouping by WBS​

This feature allows you to group your project's activities based on their WBS Codes. Select the maximum depth level you want the grouping to go through, or press the 'All' button to select all levels. We included the WBS tree (accessible through the button on the top left of Options) where you can collapse/expand and see exactly what each level includes to aid your grouping decision.

Grouping by Activity Codes​

This feature allows you to group your projects activities based on their Activity Codes. You may select multiple activity codes by clicking the arrow buttons to add Activity codes to the 'Group By' section. Use the up and down arrows within the 'Group By' section to reorder the grouping levels as needed. The above screenshot showcases adjustments to the grouping level colors.

Favorite Reports​

When clicked, the Favorite Reports button will open a small menu displaying all favorite reports and a few buttons. These buttons redirect you to the report center (maybe to adjust which reports are your favorites), add a new report, quick-save a new report with the current gantt settings, and close the menu. You can also quickly edit/duplicate/delete favorite reports with the icons in the report list.

danger

Deleting a report completely removes all traces of it from this project permanently. If you meant to just remove a report from your favorites list, you can do so by opening the report center and clicking the star next to the desired report.

Collapse/Expand One Level​

These two buttons are only enabled when a WBS or Activity code grouping is applied. These buttons make it easy to collapse or expand one grouping level at a time. The expand or collapse button will be disabled if you have fully expanded or collapsed all possible levels respectively.

PDF Export​

Download the current gantt display into a PDF. Additionally, the pdf will display the project name, update name, and update date on the top of your pdf image.
